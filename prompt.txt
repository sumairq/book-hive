# Book Hive — Production Grade Full Stack Application Prompt

You are a senior full-stack architect and staff-level engineer.

The application is a social reading tracker platform where users can:
- Track books they are reading
- Track reading progress
- Follow other users
- View activity feeds from followed users
- Write notes for books
- Control note visibility (public/private)
- Maintain reading lists
- Discover books using the Open Library API

- Next.js (App Router)
- TypeScript
- MongoDB

- Tailwind CSS
